# デザインパターン練習勉強会

```src``` 以下にデザインパターンごとにディレクトリを作成する

以下のようにフォルダを構成している

- src 
  - [ pattern name ]
    - docs
      - class.drawio
      - specification.md
    - main.ts

以下のコマンドで実行できるようにする。

```
brew install deno
deno run src/[pattern name]/main.ts
```
